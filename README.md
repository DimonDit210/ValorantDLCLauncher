# VALORANT RALPH DLC – Руководство пользователя

## 1. Приветствие

**ВНИМАНИЕ:** Использование данного скрипта осуществляется исключительно на ваш страх и риск; автор не несёт никакой ответственности за возможные последствия, включая блокировку вашего игрового аккаунта.

Этот скрипт автоматически стреляет при наведении прицела на врага. Всё, что от вас требуется – запустить скрипт, настроить горячие клавиши и играть. Ниже – только самое важное.

## 2. Обязательные требования

- **Режим экрана в игре:** Полноэкранный или оконный без полей (чтобы скрипт видел пиксели).
- **Цвет врагов:** Фиолетовая подсветка (по умолчанию в Valorant враги подсвечены фиолетовым).

## 3. Горячие клавиши и функционал

| Клавиша           | Действие |
|-------------------|----------|
| **F12**           | Включить / выключить авто-стрельбу. Заголовок окна сзтанет зелёным (вкл) или красным (выкл). |
| **END**           | Отправить «gg» в игровой чат. |
| **HOME**          | Скрыть или показать окно настроек. |
| **F4**            | Полностью закрыть скрипт. |
| **Правый Ctrl**   | Включить режим ручной подстройки задержек (Edit). |
| **Insert**        | В режиме Edit переключить, какой параметр меняем: минимальную (min) или максимальную (max) задержку. |
| **Стрелка ↑/↓**   | В режиме Edit увеличить/уменьшить выбранную задержку. |
| **Ctrl+1**        | Выбрать пресет **Vandal**. |
| **Ctrl+2**        | Выбрать пресет **Operator**. |
| **Ctrl+3**        | Выбрать пресет **Phantom**. |
| **Ctrl+4**        | Выбрать пресет **Sheriff**. |
| **Ctrl+5**        | Выбрать пресет **Bandit**. |
| **Ctrl+6**        | Выбрать пресет **Ghost**. |
| **Ctrl+7**        | Выбрать пресет **Frenzy**. |
| **Ctrl+8**        | Выбрать пресет **Bucky**. |
| **Ctrl+9**        | Выбрать пресет **Marshal**. |
| **Ctrl+0**        | Выбрать пресет **Outlaw**. |
| **Ctrl+Shift+1**  | Выбрать пресет **Guardian**. |
| **Ctrl+Alt+8**    | Сменить тип стрельбы: Auto → Single → Burst. |
| **Ctrl+Alt+9**    | Переключить режим задержки: случайная / фиксированная. |
| **Ctrl+0**        | Сбросить все параметры к текущему пресету оружия. |
| **F6**            | Показать координаты мыши и цвет пикселя под курсором (помогает при настройке цвета врага). |
| **Ctrl+I**        | Включить/выключить BunnyHop (автоматические прыжки при зажатом пробеле). |

## 4. Дополнительно

- Все настройки можно менять в окне (которое скрывается по **HOME**), но большинство действий доступны прямо с клавиатуры.
- Для пресетов предусмотрены оптимальные значения чувствительности, FOV и задержек, но вы всегда можете подстроить их вручную через GUI или горячие клавиши редактирования.
- Скрипт поддерживает компактный режим: при запуске отображается маленькое окно с названием и кнопкой развернуть (▼). Нажмите её, чтобы открыть полное меню настроек.
