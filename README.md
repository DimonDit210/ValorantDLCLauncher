# RALPH DLC – ИНСТРУКЦИЯ ДЛЯ ПОЛЬЗОВАТЕЛЯ

Спасибо за покупку! Данный продукт состоит из двух частей:  
1. **Лаунчер (`Launcher.exe`)** – отвечает за активацию ключа и автоматический запуск основного скрипта.  
2. **Автономный скрипт** – непосредственно читерский софт (триггер, BunnyHop и т.д.).

> **ВНИМАНИЕ:** скрипт и лаунчер требуют прав администратора для корректной работы!

---

## 1. ПЕРВЫЙ ЗАПУСК

- Распакуйте архив в любую папку (желательно исключить её из антивируса).  
- Запустите **`Ralph DLC.exe`** от имени администратора (ПКМ → «Запуск от имени администратора»).  
- В поле `XXXX-XXXX-XXXX-XXXX` введите полученный при покупке лицензионный ключ.  
- Нажмите **`Activate`**.  
- Если ключ валиден, лаунчер запустит основной скрипт. Ничего больше нажимать не нужно.

---

## 2. ЕСЛИ ЛАУНЧЕР НЕ ЗАПУСКАЕТСЯ ИЛИ ОШИБКА

- Убедитесь, что запускаете **от имени администратора**.  
- Проверьте подключение к интернету (для проверки обновлений и активации).  
- Добавьте папку с лаунчером в исключения антивируса (особенно Windows Defender).  
- Если появляется ошибка о отсутствии DLL, установите **Microsoft Visual C++ Redistributable** (скачать с сайта Microsoft).

---

## 3. ОБНОВЛЕНИЕ

- Лаунчер автоматически проверяет наличие новой версии при запуске.  
- Если обновление найдено, нажмите **`Update to X.X`**.  
- Лаунчер скачает новую версию, распакует и заменит себя. После этого перезапустится.  
  *(вводить ключ заново не требуется)*

---

## 4. УПРАВЛЕНИЕ СКРИПТОМ (ИГРОВЫЕ ФУНКЦИИ)

После успешной активации запускается основной скрипт. По умолчанию он создаёт два окна:  
- **Главное окно** – все настройки.  
- **Маленькое окно** – показывает статус (ON/OFF).

### Горячие клавиши  
*(работают ТОЛЬКО когда скрипт активен и VALORANT в фокусе)*

| Комбинация | Действие |
|------------|----------|
| `F12` | Включить / выключить триггер (звуковое подтверждение) |
| `END` | Отправить «gg» в чат |
| `HOME` | Скрыть / показать главное окно настроек |
| `F4` | Полностью закрыть скрипт |
| `Правый Ctrl` | Режим редактирования задержек (после включения стрелки вверх/вниз меняют задержки) |
| `Insert` | Переключить редактируемый параметр (min/max) при активном режиме редактирования |
| `Ctrl+1…0` | Выбор пресета оружия (1 – Vandal, 2 – Operator, 3 – Phantom, … 0 – Outlaw) |
| `Ctrl+Shift+1` | Пресет Guardian |
| `Ctrl+Колёсико мыши` | Переключение пресетов вверх/вниз |
| `Ctrl+Alt+8` | Переключить тип стрельбы (Auto / Single) |
| `Ctrl+Alt+9` | Переключить режим задержки (случайная / фиксированная) |
| `Ctrl+I` | Включить / выключить BunnyHop (автопрыжки при зажатом пробеле) |

### Клавиша активации триггера (по умолчанию – `LALT`)

- **Режим Hold:** зажимаете `LALT` – триггер активен, отпускаете – выключен.  
- **Режим Toggle:** один раз нажали `LALT` – триггер включен до следующего нажатия.  

Клавишу и режим можно изменить во вкладке **«Дополнительно» → «Клавиша активации»**.

## 5. НАСТРОЙКИ ТРИГГЕРА

Все настройки сохраняются в файл `config.ini`.  
Рекомендуемые значения (заводские пресеты уже настроены оптимально):

| Параметр | Значение | Описание |
|----------|----------|----------|
| **Допуск цвета** | 100 | чем выше, тем хуже различает врага |
| **FOV ширина/высота** | 7–11 | область поиска вокруг прицела |
| **Мин. пикселей** | 1 | сколько совпадающих пикселей нужно для выстрела |
| **Задержка реакции** | случайная 0.0001–0.0005 с | регулирует скорость стрельбы |

Пресеты оружия (Vandal, Operator и т.д.) автоматически меняют FOV и задержки под конкретное оружие.  
Для тонкой подстройки используйте вкладки **«Стрельба»**, **«Humanizer»**, **«Дополнительно»**.

## 6. УСТРАНЕНИЕ ПРОБЛЕМ

| Проблема | Решение |
|----------|---------|
| Триггер не стреляет | проверьте, что VALORANT запущен, вы нажали `F12` (статус ON), и зажата клавиша активации |
| Скрипт не видит врага | уменьшите допуск цвета (например, до 50) или увеличьте FOV |
| Сильная нагрузка на процессор | уменьшите FOV, включите ограничение очереди для Vandal/Phantom |
| Лаунчер не обновляется | проверьте интернет, временно отключите антивирус |
